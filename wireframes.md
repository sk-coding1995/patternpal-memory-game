# Wireframes – PatternPal Memory Game

## Overview
This file documents the layout planning for the PatternPal Memory Game. The aim of the wireframe was to create a simple and clear game interface that is easy for the user to understand and interact with.

## Planned layout

The planned layout included:

- A clear main heading at the top of the page
- A move counter underneath the heading
- A restart button placed near the top for easy access
- A centred game grid containing the cards
- Clear spacing between cards so the board remains readable

## Design reasoning

The layout was designed to keep the main game board as the focus of the page. Important information such as the title, move counter, and restart button was placed above the grid so the player can immediately understand the purpose of the page and control the game easily.

A single-page layout was chosen to keep navigation simple and intuitive. This supports the user experience by reducing unnecessary complexity and allowing the player to focus entirely on the game.

## Implementation notes

The implemented version of the project follows this planned structure closely. The game includes:

- A heading showing the PatternPal Memory Game title
- A move counter
- A restart button
- A centred responsive card grid

The final implementation remained close to the original planned layout because the design proved clear and effective during development.
